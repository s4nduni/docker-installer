# Docker Installer

This will install Docker engine in any Ubuntu desktop.

## Installation

Clone this repo.

```bash
git clone https://github.com/s4nduni/docker-installer.git
```

## Usage

Go to directory.

```
cd docker-installer
```

Install docker.

```
./install
```

Uninstall docker.

```
./uninstall
```
Please restart your system after running the program.
